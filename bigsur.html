Hello Guys, So you want to install Mac Os Big Sur on your Windows PC/laptop, you have come to the right place.

Requirements :

Flash drive

Big Sur.raw

Propertree

Paragon

Explorer++

balenaEtcher

Kexts

Pre-Made-EFI

Process :
So once you have all the requisites let’s proceed with the installation.
1. Plugin the USB Pendrive.
2. Now Extract the Olarila Mac os Big Sur.raw from above link File using Winrar
3. Once Finished Extracting, Open Balena etcher.
4. Click On Flash from File and select the raw file.
5. Then select your USB and click on flash.
6. While it’s flashing, let’s create the EFI folder.
7. Now you can download a premade EFI folder that fits your hardware from the above links. Or Create an EFI folder yourself.
8. For creating an EFI folder you need to open the Opencore Pkg that we downloaded from the above links
Navigate to Opencore/x64/EFI/OC. 9. Here we need to place some files lets start with the ACPI folder.
10.Go to this link select your intel/AMD processor from configs.
11.Then go to ACPI select prebuilt SSDTs and download the SSDTs required for your processor.
12.Once downloaded place those into the ACPI folder.
13.Now go to the driver folder and here delete all the files except Openruntime and open canopy.
14.Also, open the apple support pkg that we downloaded from the link above. And copy vbox to drivers folder.
15.Now open the kext folder and place all your required kexts.The kext that are mostly required are.
a. Lilu
b. Whatevergreen
c. Virtual SMC
d. Apple ALC
e. Ethernet kext (realtek or intel do a google search)
19. Now go to tools folder and delete all the files from it.
20. Now go back to the opencore main file and here open the docs folder
21. Copy the sample.plist from there to EFI/OC and rename it to config.plist.
22. Now open propertree.bat that we downloaded from the above link .
23. Then in propertree go to file/open and select the config.plist file that we just created in EFI/OC folder.
24. Again go to file then click on OC snapshot and this time select the OC folder itself and click on select folder.
25. Now you will need to do some changes to your config.plist, head to the opencore website and there configure your config .plist according to the intel processor you have.
26. Once done save the config.plist.
27. Now hoping that your USB has finished flashing on balena etcher , install and open paragon partition manager that we downloaded from the above links.
28. Then select your usb and you will see a 200MB partition select the partition on the left side you will see the change letter option , select any letter you want and click on change now wait for few minutes.
29. Once you have assigned a new letter open explorer++ as admin.
30. Here select your EFI partition that we just assigned a letter, then delete the efi folder present here and paste the efi folder that we create above.
Now reboot from usb and Voila mac os installer should boot up.
