Installing Chrome OS on your windows PC or laptop

Requirements :

Linux Mint

Rufus

Brunch

install.sh

Chrome OS Download

Process :
To start off, you are gonna need a USB pen drive and an ISO file of LINUX Mint. To download the ISO file, go to the link (URL here) and select option ‘cinnamon’. Select a server among the lots that appears and download the ISO file. For the procedure, you will also need to download RUFUS. Once downloaded, open the software and select the LINUX mint IOS file and click start. The following dialogue box that appears will have two options. Select “write in ISO image mode” and click ok. Here, the ISO file starts flashing in the USB which will take not more than 10 minutes. Once the flashing is done, reboot your computer and make sure you go to your boot menu. From the options select the USB in UEFI mode. Next you will see LINUX start grub bootloader. Select the first option and your system will start the boot up. Once you are booted into LINUX mint, connect to the internet and download the chrome OS recovery from (URL here). Among the many options you can see on screen, select rummus and download its latest version. For the process, you will also need to save (URL here) this install.sh file in your PC. After saving the file, move along to download brunch from (URL here). Brunch basically is a bootloader that helps to oot chrome in normal PCs Now open any hard disk, create a new folder and name it chrome OS. Move the downloaded install.sh file in the chrome OS folder along with brunch and chrome OS after their extractions. Next step is to rename the Chrome OS file to “Rummus recovery” Now open Gparted from the system applications. In there, you will find all the disks on your right panel. The disk that you need to install is “/dev/ssd”. This hard disk will have dummy files. Now open the file install.sh with a text editor where you will find “/dev/sda” as the default hard disk. Change it to “/dev/ssd” and save the file. Right click in the chrome OS folder and select ‘Open in terminal’ option. You will see a black window opening where you will have to type “sudo sh install.sh” and press enter. The window will start reading the packages and updates of chrome OS and will start installing in the selected hard disk. After a while, you will see a prompt asking if you want to continue. Type ‘yes’ and press enter thereby starting the flash. This will take around 10-15 minutes. Once the flashing is done, unplug the USB and restart your computer. After restarting, select the hard disk in which you installed chrome OS and make sure it is in UEFI mode. As it is booting up for the first time, it will take a little while. Once booting is done, sign into your chromebook and follow the dialogue boxes to set up chrome. Once signed in, your PC will work as Android. Why install Chrome OS? If you have an old PC or laptop and you want to use it for light browsing, you won’t need any powerful hardware with chrome OS installed in your hard disk. One more benefit of having the chrome OS is that you can turn on LINUX beta from settings which will allow you to install the LINUX apps.
